# 提案
## Why(提案動機)?
> 線上多人互動教室或線上會議室已經有許多實際應用案例，然而許多控制介面仍然侷限於傳統的滑鼠控制，並不符合人類自然表達的方式，本組希望藉由機器學習識別影像串流中的手勢來發出控制指令，例如，靜音，取消靜音，切換投影片等，以協助**教室/會議管理者** 管理線上**虛擬教室/會議室**
